### The 'Old' Folder

for stuff I want to keep as part of the project, juuuuuuuuust in case it comes in handy someday to remember something, but it is NOT to be referenced by ai agents. It's just a little ocd time capsule.

when you add something to it, first unzip the old.rar archive, delete old.rar, then highlight all files in here (including the new one(s) you just added) and rezip them as old.rar. this way there's only two things in this folder: the rar that contains all the old note files/assets, and the readme for old folder.
